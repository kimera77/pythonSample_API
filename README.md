# Star Wars API Sample

A simple Python notebook that demonstrates how to interact with the Star Wars API (SWAPI) to search for characters and planets.

## Features

- Fetch Star Wars character data from the API
- Search characters by name using multiple methods (strings, objects, and dataframes)
- Search planets by terrain type and name
- Interactive UI built with Gradio

## Requirements

- Python 3.x
- requests
- gradio
- pandas

## Installation

Install the required packages:

```bash
pip install requests gradio pandas
```

## Usage

1. Open `app.ipynb` in Jupyter Notebook or VS Code
2. Run all cells sequentially
3. Use the Gradio interface to search for planets by terrain type or name

## API Reference

This project uses the [SWAPI](https://swapi.info/) - Star Wars API to fetch data about characters and planets from the Star Wars universe.
